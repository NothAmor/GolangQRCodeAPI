NothAmor QR Code API Service

使用方法:
访问链接: /qr
参数说明:
data 二维码内包含的数据
size 二维码图片的大小, size参数要求必须为整数，且小于等于1000
type 二维码返回参数的格式, 可以为空, 默认直接返回图片
type的允许值: image, json (image为直接返回图片，json则为返回json串)
json返回值示例: {"code":200,"msg":"Success","data":{"image":base64}}
其中image值是图片的base64编码
使用示例: /qr?data=https://www.nothamor.cn&size=100&type=image
